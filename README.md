**Wordly** is a fun, original word-guessing game inspired by the classic Wordle style. Guess the secret 5-letter word in 6 tries and explore multiple unique categories full of creative, made-up words.  

---

## Features

- Multiple categories to choose from:
  - Fantasy Names  
  - Planets  
  - Creatures  
  - Foods  
  - Colors  
  - Gadgets  
  - Vehicles  
  - Sports  
  - Animals  
  - Mystic  

- Hundreds of unique words per category  
- Standard Wordle mechanics with 6 guesses per word  
- Visual feedback for each guess:
  - **Green**: correct letter in the correct position  
  - **Yellow**: correct letter in the wrong position  
  - **Gray**: letter not in the word  
- Fully playable in any modern web browser
